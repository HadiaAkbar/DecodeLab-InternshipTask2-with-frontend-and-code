# Iris AI Classifier 🌸

An intelligent iris species classifier that blends the precision of machine learning with the beauty of botanical design. Built with a focus on **Botanical Elegance**, this application provides a delightful and accessible way to explore the classic Iris dataset.

## ✨ Features

- **Species Prediction**: Input sepal and petal measurements to identify Iris Setosa, Versicolor, or Virginica.
- **Botanical Design**: A biophilic interface featuring organic shapes, soft textures, and a calming purple-and-sage palette.
- **Data Visualization**: Interactive charts and botanical cards that bring machine learning results to life.
- **Responsive Interface**: Fully optimized for both desktop and mobile devices.
- **Accessible AI**: Designed to make technical predictions feel natural and approachable.

## 🛠️ Tech Stack

### Frontend
- **Framework**: [React 19](https://react.dev/) with [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with [Shadcn UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)

### Backend
- **Server**: [Express](https://expressjs.com/)
- **Validation**: [Zod](https://zod.dev/)

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [pnpm](https://pnpm.io/) (v10 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/HadiaAkbar/DecodeLab-InternshipTask2-with-frontend-and-code.git
   cd DecodeLab-InternshipTask2-with-frontend-and-code
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm dev
   ```

4. Build for production:
   ```bash
   pnpm build
   pnpm start
   ```

## 🎨 Design Philosophy

The project follows the **Botanical Elegance** approach, which honors the subject matter—the iris flower. Key principles include:
- **Organic Geometry**: Curved shapes and flowing transitions instead of rigid grids.
- **Botanical Authenticity**: Using actual iris imagery and nature-inspired color palettes.
- **Tactile Warmth**: Soft shadows and gentle gradients for a human-centric feel.
